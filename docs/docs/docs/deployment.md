# Deployment — Tweet Forwarder Bot

## Overview
Instructions for deploying the bot safely and efficiently for real-time forwarding.

## Steps
1. Clone the repository
2. Install dependencies from requirements.txt
3. Set environment variables in `.env`
4. Run the bot: `python main.py`
5. Ensure logs and monitoring are active (feature3.md, docs/monitoring.md)

## Notes
- Designed to support 2–5 second tweet forwarding
- Can be deployed on cloud servers for high availability
- Multi-account and multi-channel ready
