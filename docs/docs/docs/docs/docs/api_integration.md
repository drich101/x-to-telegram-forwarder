# API Integration — Tweet Forwarder Bot

## Overview
Guidelines for integrating with X (Twitter) API and Telegram Bot API.

## Steps
1. Obtain X API Bearer Token
2. Obtain Telegram Bot Token
3. Configure tokens in `.env`
4. Test API connection with sample requests
5. Monitor rate limits and errors

## Notes
- Proper integration ensures near real-time forwarding (2–5 seconds)
- Supports multi-account and multi-channel forwarding
- Prepares for future Web3/Base upgrades
