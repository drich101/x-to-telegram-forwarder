# Tweet Filtering Guide — Tweet Forwarder Bot

## Overview
Instructions for setting up automated filtering to forward only relevant tweets.

## Steps
1. Define keywords, hashtags, or account lists in a configuration file
2. Set filters in bot environment or config
3. Test filtering with sample tweets
4. Adjust filters based on feedback

## Notes
- Works with multi-account and multi-channel forwarding
- Ensures ultra-fast forwarding (2–5 seconds) for relevant content
- Prepares bot for scalable Web3/Base Telegram communities
