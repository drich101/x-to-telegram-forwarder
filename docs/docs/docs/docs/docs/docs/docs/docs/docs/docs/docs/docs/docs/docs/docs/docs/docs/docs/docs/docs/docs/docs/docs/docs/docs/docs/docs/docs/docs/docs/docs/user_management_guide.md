## Overview
This guide explains how admins can manage users, monitor trials, and control paid subscriptions via the dashboard.

## Features
- **View Users:** List of all trial and paid users with subscription status
- **Add Users:** Manually add users or assign trial periods
- **Deactivate Users:** Block users who violate rules or cancel subscriptions
- **Track Subscriptions:** Monitor start and expiry dates for trials and paid plans
- **Notifications:** Send reminders or alerts to users about trial expiry

## Admin Steps
1. Log in to the admin panel
2. Access User Management Dashboard
3. Filter users by subscription type or status
4. Add, deactivate, or modify user information
5. Export user data for analytics or auditing

- Integrates with Subscription Validation Middleware (feature34.md)
