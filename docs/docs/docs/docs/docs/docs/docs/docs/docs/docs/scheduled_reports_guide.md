# Scheduled Reports Guide — Tweet Forwarder Bot

## Overview
Instructions for configuring scheduled analytics reports for admins.

## Steps
1. Enable scheduled reporting in bot configuration
2. Set reporting interval (daily, weekly, etc.)
3. Include metrics: forwarded tweets, latency, errors
4. Send reports to admin Telegram accounts
5. Test and adjust report settings

## Notes
- Supports ultra-fast forwarding (2–5 seconds)
- Complements analytics (feature14.md) and monitoring (docs/monitoring.md)
- Ensures professional and trackable bot performance in Web3/Base communities
