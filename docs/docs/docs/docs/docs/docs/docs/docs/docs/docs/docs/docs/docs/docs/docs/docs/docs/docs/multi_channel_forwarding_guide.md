# Multi-Channel Forwarding Guide — Tweet Forwarder Bot

## Overview
Instructions for configuring the bot to forward tweets to multiple Telegram channels or groups.

## Steps
1. Access the bot admin dashboard
2. Add Telegram channels/groups to forwarding list
3. Map channels to priority or filter rules if needed
4. Test forwarding to ensure ultra-fast delivery (2–5 seconds)
5. Adjust channel mapping based on traffic or performance

## Notes
- Works with Priority Forwarding (feature24.md) and Custom Filtering (feature25.md)
- Supports scalable and professional Web3/Base Telegram communities
- Ensures consistent forwarding across multiple channels/groups
