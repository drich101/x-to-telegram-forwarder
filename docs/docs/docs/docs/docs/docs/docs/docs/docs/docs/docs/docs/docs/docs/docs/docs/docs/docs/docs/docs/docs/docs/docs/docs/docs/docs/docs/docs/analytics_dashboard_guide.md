## Overview
This guide explains how admins can use the Analytics Dashboard to monitor bot performance across all platforms (X, Farcaster, Calaxy).

## Key Metrics
- **Forwarded Posts:** Count per platform and per channel
- **Latency:** Average detection-to-forwarding time
- **Retries & Failovers:** Number of failed forwards and retry attempts
- **Active Users:** Trial vs Paid subscriber counts
- **Multi-Channel Performance:** Metrics per Telegram channel

## Admin Usage
1. Access dashboard via secure admin panel
2. View real-time graphs and metrics
3. Drill down into historical logs
4. Configure alerts for latency > 5s or failed forwards
5. Export reports for analytics or auditing
