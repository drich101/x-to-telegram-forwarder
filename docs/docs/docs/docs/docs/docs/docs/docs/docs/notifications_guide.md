# Notifications Guide — Tweet Forwarder Bot

## Overview
Instructions for configuring admin notifications for errors, failures, and performance metrics.

## Steps
1. Enable notifications in admin dashboard (feature15.md)
2. Configure alert types: errors, failures, latency thresholds
3. Set admin Telegram accounts for notifications
4. Test notifications with sample events
5. Monitor and adjust thresholds as needed

## Notes
- Supports ultra-fast forwarding (2–5 seconds)
- Works with logging, monitoring, and analytics
- Ensures proactive bot maintenance in Web3/Base communities
