# Priority Forwarding Guide — Tweet Forwarder Bot

## Overview
Instructions for configuring priority forwarding for selected accounts or hashtags.

## Steps
1. Access the bot admin dashboard
2. Add Twitter/X accounts or hashtags to the priority list
3. Assign priority levels (e.g., high, medium, low)
4. Test forwarding speed to ensure 1–2 second delivery
5. Monitor and adjust based on traffic and latency

## Notes
- Works with retry mechanism (feature10.md) and latency monitoring (feature21.md)
- Ensures ultra-fast forwarding for high-value tweets
- Supports scalable and professional Web3/Base Telegram communities
