## Overview
This guide explains how admins can configure content filters and moderation rules to control what is forwarded to subscribers.

## Features
- **Keyword Filtering:** Block or allow posts containing specific words
- **Hashtag Filtering:** Forward posts only if certain hashtags are present
- **Media Type Filtering:** Control which types of media are forwarded (images, videos, GIFs, attachments)
- **Blacklists & Whitelists:** Manage specific users, accounts, or content sources
- **Moderation Alerts:** Notify admins of flagged content

## Admin Steps
1. Log in to the admin panel
2. Access the Filtering & Moderation Dashboard
3. Configure keyword, hashtag, or media filters
4. Manage blacklists and whitelists
5. Enable moderation alerts for critical content
6. Monitor logs for blocked or forwarded posts
