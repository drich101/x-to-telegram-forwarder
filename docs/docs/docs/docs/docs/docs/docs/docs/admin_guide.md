# Admin Guide — Tweet Forwarder Bot

## Overview
Instructions for using the admin dashboard (feature15.md) to manage accounts, channels, and forwarding rules.

## Steps
1. Access dashboard via secure login
2. Add/remove Twitter accounts to monitor
3. Configure Telegram channels for forwarding
4. Set filtering and forwarding rules
5. Monitor performance and logs

## Notes
- Supports ultra-fast forwarding (2–5 seconds)
- Works with logging, monitoring, and analytics features
- Prepares bot for large-scale Web3/Base Telegram communities
