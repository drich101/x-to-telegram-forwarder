# Analytics Dashboard Guide — Tweet Forwarder Bot

## Overview
This guide explains how to use the Analytics Dashboard to monitor bot performance and subscriber activity.

## Available Metrics
- Total forwarded tweets per account
- Total forwarded tweets per Telegram channel/group
- Average forwarding latency (target: 2–5 seconds)
- Failed forwarding attempts
- Active trial users
- Active paid subscribers

## Steps to Use
1. Access the Admin Dashboard
2. Open the Analytics section
3. Select account or channel to view metrics
4. Review latency and performance charts
5. Export reports if needed
