# Analytics & Performance Dashboard Guide — Base Ecosystem Forwarder Bot

## Overview
This guide explains how admins can use the Analytics & Performance Dashboard to monitor bot activity, latency, failures, media handling, and subscriptions.

## Features
- **Platform Metrics:** Track posts forwarded per platform (X, Farcaster, Calaxy)
- **Latency Monitoring:** Measure time from post detection to forwarding
- **Failure Tracking:** Detect failed or retried posts
- **Media Analytics:** Monitor images, videos, attachments forwarded
- **Subscription Analytics:** Track trial, active, and expired users
- **Reports:** Export reports for review and auditing

## Admin Steps
1. Log in to the admin panel
2. Access the Analytics & Performance Dashboard
3. Filter metrics by platform, media type, or date
4. Monitor latency, failures, and retry rates
5. Export reports as needed
6. Adjust settings or priorities based on performance insights

## Notes
- Works with Unified Social Stream (feature43.md), Multi-Channel Forwarding (feature35.md), and Retry System (feature37.md)
- Critical for maintaining 2–5 second forwarding goal
- Helps identify and resolve bottlenecks
