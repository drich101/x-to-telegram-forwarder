## Overview
This guide explains how the bot detects post languages and optionally translates content before forwarding.

## Features
- **Language Detection:** Automatically detect the language of incoming posts
- **Forwarding:** Forward posts in original language
- **Optional Translation:** Translate posts to target language if enabled
- **Analytics Logging:** Record detected languages for performance and user metrics
- **Admin Settings:** Enable/disable translation, set default language

## Admin Steps
1. Log in to the admin panel
2. Access Multi-Language Settings
3. Enable automatic language detection
4. Configure optional translation settings
5. Monitor language analytics for user engagement

## Notes
- Integrates with Unified Social Stream (feature43.md) and Multi-Channel Forwarding (feature35.md)
- Supports multi-platform content (X, Farcaster, Calaxy)
- Critical for serving international audiences
