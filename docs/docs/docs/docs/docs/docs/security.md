# Security — Tweet Forwarder Bot

## Environment Variables
- Never commit your `.env` with real tokens
- Use `.env.example` as a placeholder

## API Keys
- Keep Telegram Bot Token and X (Twitter) API Bearer Token private
- Rotate keys periodically for security

## Access Control
- Admin-only commands (feature2.md)
- Limit access to sensitive features

## Notes
- Ensures bot safety while forwarding tweets in near real-time (2–5 seconds)
- Prepares bot for secure Web3/Base integration
